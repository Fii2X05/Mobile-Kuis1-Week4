Program ini dibuat menggunakan bahasa Dart untuk menghitung total belanja dan diskon pada sebuah toko.



Program menggunakan 3 digit terakhir NIM yaitu 120 yang disimpan dalam variabel nilaiUnikNIM. Nilai ini dimasukkan ke dalam List<double> hargaBarang sebagai elemen tambahan sehingga mempengaruhi total belanja.



Total belanja dihitung menggunakan fungsi hitungTotal dengan perulangan for-in. Setelah total diperoleh, program menentukan diskon berdasarkan kategori berikut:



Total > 200000 mendapatkan diskon 10%

Total 100000 - 200000 mendapatkan diskon 5%

Selain itu tidak mendapatkan diskon



Karena nilai unik NIM dimasukkan ke dalam daftar harga, maka total belanja akan berbeda untuk setiap mahasiswa sehingga hasil perhitungan bersifat personal.
